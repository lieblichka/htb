### Для взаимодействия с базой Redis необходимо использовать 

		redis-cli

	# INSTALL CLIENT
	Установка клиента

		@ sudo pacman -S redis 


### ENUMERATION

	-  Подключение к хосту, по умолчанию используется хост 127.0.0.1 и порт
	6379

		@ redis-cli -h <ip> # подключение к хосту
			
			10.129.130.102:6379>

	- После успешного подключения к серверу Redis мы должны увидеть 
		приглашение в терминале
		
		
		10.129.130.102:6379>
			
		~ ping # проверить подключение к базе
				
			PONG

		~ info # узнать статистику сервера redis

			# Serve
			redis_version:5.0.7
			redis_git_sha1:00000000
			redis_git_dirty:0
			redis_build_id:66bd629f924ac924

			[...SNIP...]
			
			# Keyspace
			db0:keys=4,expires=0,avg_ttl=0

	- Здесь ключевое слово пространство Keyspace. В нашем случае под Ключевое 
	пространство мы видим, что существует только одна база данных с индексом. 0

		~ SELECT 0 # используется для выбора нужной базы

			OK
			10.129.130.102:6379> SELECT 0

		~ KEYS * # команда для получения всех ключей базы 

			OK
			10.129.130.102:6379> KEYS *

		~ GET flag # узнать что содержится по ключу flag
				
			"03e1d2b376c37ab3f5319922053953eb"

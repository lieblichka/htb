*** JAVASCRIPT

	
	JavaScript — один из самых используемых языков в мире. 		

		https://en.wikipedia.org/wiki/JavaScript

	Он в основном используется для веб-разработки и мобильной разработки.
	JavaScriptобычно используется во внешнем интерфейсе приложения, которое
	должно выполняться в браузере. Тем не менее, существуют реализации
	внутреннего JavaScript, используемые для разработки целых веб-приложений,
	например NodeJS .

		https://nodejs.org/en/about/

	Пока HTMLа также CSSв основном отвечают за внешний вид веб-страницы,
	JavaScriptобычно используется для управления любой функциональностью,
	которая требуется интерфейсной веб-странице. Без JavaScript, веб-страница
	будет в основном статической и не будет иметь большого количества функций
	или интерактивных элементов.

	### Usage
	Применение
		
		Большинство распространенных веб-приложений сильно зависят от
		JavaScriptдля управления всеми необходимыми функциями на веб-странице,
		такими как обновление представления веб-страницы в режиме реального
		времени, динамическое обновление содержимого в режиме реального времени,
		прием и обработка пользовательского ввода и многие другие потенциальные
		функции.

		JavaScriptтакже используется для автоматизации сложных процессов и
		выполнения HTTP-запросов для взаимодействия с внутренними компонентами,
		а также для отправки и извлечения данных с помощью таких технологий, как
		Ajax.
			
			https://en.wikipedia.org/wiki/Ajax_(programming)

		Помимо автоматизации, JavaScriptтакже часто используется вместе CSS, как
		упоминалось ранее, для управления продвинутой анимацией, которая была бы
		невозможна с CSSодин. Всякий раз, когда мы посещаем интерактивную и
		динамичную веб-страницу, на которой используется множество продвинутых и
		визуально привлекательных анимаций, мы видим результат активной
		JavaScriptкод, работающий в нашем браузере.

		Все современные веб-браузеры оснащены JavaScriptдвигатели, которые могут
		выполнять JavaScriptкод на стороне клиента, не полагаясь на внутренний
		веб-сервер для обновления страницы. Это делает использование
		JavaScriptочень быстрый способ быстро выполнить большое количество
		процессов.

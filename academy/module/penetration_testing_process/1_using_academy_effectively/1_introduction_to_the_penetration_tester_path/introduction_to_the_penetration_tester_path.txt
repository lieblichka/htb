*** INTRODUCTION TO THE PENETRATION TESTER PATH  
Введение в путь тестера на проникновение
	
	- Этот модуль представляет собой введение в ~путь работы~ тестировщика на 
	проникновение и общее введение в тесты на проникновение и каждый из этапов, 
	которые мы подробно рассматриваем в модулях. Мы рекомендуем начать путь с 
	этого модуля и периодически обращаться к нему по мере прохождения других 
	модулей, чтобы увидеть, как каждая тематическая область вписывается в общую 
	картину процесса тестирования на проникновение. Этот модуль также является 
	отличной отправной точкой для всех, кто плохо знаком с HTB Academy или 
	отраслью.
		
		~.L.I.N.K.~ https://academy.hackthebox.com/paths/jobrole 

	Этот путь предназначен для начинающих тестировщиков на проникновение из всех 
	слоев общества и опытных пентестеров, желающих повысить квалификацию в 
	определенной области, стать более разносторонним или изучить вещи с другой 
	точки зрения. Этот путь охватывает основные концепции, необходимые для 
	успешного прохождения внешних тестов на проникновение, внутренних тестов на 
	проникновение (как сетевых, так и Active Directory) и оценок безопасности 
	веб-приложений. В каждом модуле мы глубоко погружаемся в специализированные 
	методы, методологии и инструменты, необходимые для успеха в тестировании на 
	проникновение. Путь отправляет студентов в практическое путешествие по всем 
	этапам теста на проникновение, от разведки и подсчета до документации и 
	отчетности, кульминацией которого является смоделированный модуль проверки 
	на проникновение. Студенты, полностью завершившие этот путь, будут вооружены 
	практическими навыками и мышлением, необходимыми для выполнения 
	профессиональных оценок безопасности в реальных сетях на уровне от 
	базового до среднего. Каждый из наших модулей глубоко погружается в «почему», 
	стоящие за проблемами и тактиками, которые мы представляем, и является не 
	просто учебным пособием по работе с инструментами «укажи и щелкни». Мы 
	рассказываем истории и сценарии, основанные на нашем реальном опыте проведения 
	оценок безопасности для клиентов во всех вертикалях, а также для местных и 
	федеральных органов власти. 

	### HTB Academy Learning Philosophy
	Философия обучения Академии HTB
		
		- Наша философия заключается в том, чтобы «учиться на практике», следуя 
		подходу, основанному на оценке рисков, с упором на практическое обучение 
		и законное и этичное использование наших навыков. Мы стремимся научить 
		наших студентов «почему» уязвимости и тому, как обнаруживать, использовать, 
		исправлять, обнаруживать и предотвращать недостатки, чтобы создать 
		всесторонних профессионалов, которые могут передать эти всеобъемлющие 
		знания и образ мышления своим нынешним/будущим. клиентов или работодателей, 
		чтобы помочь им защитить своих людей, технологии и миссии от современных 
		киберугроз.

	### Ethical and Legal Considerations
	Этические и юридические аспекты
		
		- Если вы готовы попрактиковаться на реальных объектах, вы можете 
		получить дополнительную практику, участвуя в программах вознаграждения 
		за обнаружение ошибок, организованных такими организациями, как ~HackerOne~ 
		и ~Bugcrowd~. Благодаря этим организациям по вознаграждению за обнаружение 
		ошибок вы можете участвовать в мероприятиях по тестированию веб-приложений 
		от множества различных компаний, предлагающих программу вознаграждения за 
		обнаружение ошибок. Имейте в виду, что каждая из этих программ имеет свою 
		область применения и правила участия, поэтому ознакомьтесь с ними, прежде 
		чем начинать какие-либо действия по тестированию. Большинство этих 
		программ не поддерживают автоматическое сканирование, что делает их 
		отличным способом попрактиковаться в сборе информации и ручном тестировании 
		веб-приложений.

	### Penetration Tester Path Syllabus
	Программа тестирования на проникновение

		- Путь имитирует тест на проникновение против компании Inlanefreight, 
		разбитый на различные этапы, охватывая основные концепции и инструменты, 
		которые сделают нас выделяющимися как тестировщиками на проникновение. 
		Путь завершается углубленным модулем по критически важным навыкам 
		межличностного общения, таким как ведение заметок, организация, 
		документирование, отчетность и общение с клиентами, а затем полноценный 
		имитационный тест на проникновение, чтобы отработать все наши навыки в 
		одной большой моделируемой корпоративной сети. Модули, составляющие путь, 
		расположены следующим образом:

		# Introduction
		
			1. Penetration Testing Process
			2. Getting Started

		# Reconnaissance, Enumeration & Attack Planning
			
			3. Network Enumeration with Nmap
			4. Footprinting
			5. Information Gathering - Web Edition
			6. Vulnerability Assessment
			7. File Transfers
			8. Shells & Payloads
			9. Using the Metasploit Framework

		# Exploitation & Lateral Movement 
			
			10. Password Attacks
			11. Attacking Common Services
			12. Pivoting, Tunneling, and Port Forwarding
			13. Active Directory Enumeration & Attacks

		# Web Exploitation 

			14. Using Web Proxies
			15. Attacking Web Applications with Ffuf
			16. Login Brute Forcing
			17. SQL Injection Fundamentals
			18. SQLMap Essentials
			19. Cross-Site Scripting (XSS)
			20. File Inclusion
			21. File Upload Attacks
			22. Command Injections
			23. Web Attacks
			24. Attacking Common Applications

		# Post-Exploitation 
			

			25. Linux Privilege Escalation
			26. Windows Privilege Escalation
			
		# Reporting & Capstone 
			
			27. Documentation & Reporting
			28. Attacking Enterprise Networks
		
                                                                
		- После завершения этого пути мы рекомендуем учащимся работать над 
		специализацией, будь то Active Directory, Web или реверс-инжиниринг. Мы 
		должны постепенно продолжать наращивать свои навыки во всех областях, 
		чтобы стать как можно более всесторонними, стремясь к знаниям экспертного 
		уровня по крайней мере в одной дисциплине. Для тех, кто еще не чувствует 
		себя достаточно уверенно, чтобы пройти этот путь работы, мы рекомендуем 
		пройти все модули в Information Security Foundations ~Skill Path~ , который 
		поможет получить предварительные знания, необходимые для успешного 
		прохождения пути Penetration Tester. Лучше всего брать модули в Пути 
		тестера на проникновение по порядку, так как концепции строятся друг на 
		друге. На протяжении всего пути мы постоянно повторяем концепции и 
		представляем вещи по-разному, чтобы дать студентам больше практики и 
		сценариев для расширения их знаний в определенной области. Например, 
		боковое движение и мародерство являются жизненно важными навыками для 
		достижения успеха в тестировании на проникновение. У нас нет отдельных 
		модулей для каждой из этих фаз, потому что мародерство и боковое 
		перемещение можно рассматривать как повторяющиеся процессы, к которым мы 
		будем возвращаться много раз во время оценки. Вместо этого мы представляем 
		тактику, методы и процедуры для бокового перемещения и показываем широкий 
		спектр сценариев, в которых мы можем извлечь выгоду из сильных навыков 
		мародерства/пост-эксплуатации.

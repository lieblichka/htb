*** Cascading Style Sheets (CSS)

	CSS (каскадные таблицы стилей) — это язык таблиц стилей, используемый наряду
	с HTML для форматирования и установки стиля элементов HTML. Как и HTML,
	существует несколько версий CSS, и каждая последующая версия представляет
	новый набор возможностей, которые можно использовать для форматирования
	элементов HTML. Браузеры обновляются вместе с ним для поддержки этих новых
	функций. 
		
		https://www.w3.org/Style/CSS/Overview.en.html
	
	# Example
	Пример
		
		На фундаментальном уровне CSS используется для определения стиля каждого
		класса или типа HTML-элементов (т. е. bodyили же h1), так что любой
		элемент на этой странице будет представлен так, как определено в файле
		CSS. Это может включать семейство шрифтов, размер шрифта, цвет фона,
		цвет и выравнивание текста и многое другое.
		~~~~		
			body {
			  background-color: black;
			}

			h1 {
			  color: white;
			  text-align: center;
			}

			p {
			  font-family: helvetica;
			  font-size: 10px;
			}

		~~

		Как упоминалось ранее, именно поэтому мы можем установить уникальные
		идентификаторы или имена классов для определенных элементов HTML, чтобы
		мы могли позже обращаться к ним в CSS или JavaScript, когда это
		необходимо.

	### Syntax
	Синтаксис
		
		CSS определяет стиль каждого HTML-элемента или класса в фигурных
		скобках. {}, в котором свойства определяются своими значениями (т.е.
		element { property : value; }).

		Каждый элемент HTML имеет множество свойств, которые можно задать с
		помощью CSS, например height, position, border, margin, padding, color,
		text-align, font-sizeи сотни других свойств. Все это можно комбинировать
		и использовать для создания визуально привлекательных веб-страниц.

		CSS можно использовать для продвинутой анимации для самых разных целей,
		от перемещения элементов до продвинутой 3D-анимации. Многие свойства CSS
		доступны для анимации, например @keyframes, animation,
		animation-duration, animation-direction, и многие другие. Вы можете
		прочитать и попробовать многие из этих свойств анимации здесь .
			
			https://www.w3schools.com/css/css3_animations.asp

			элементов HTML или для создания расширенной анимации на основе
			нажатий клавиш или положения курсора мыши.

	### Frameworks
	Фреймворки
		
		Многие могут подумать, что CSS сложно разрабатывать. Напротив, другие
		могут возразить, что неэффективно вручную устанавливать стиль и дизайн
		всех элементов HTML на каждой веб-странице. Вот почему было представлено
		множество фреймворков CSS, которые содержат набор таблиц стилей и
		дизайнов CSS, чтобы сделать создание красивых HTML-элементов намного
		быстрее и проще.

		Кроме того, эти платформы оптимизированы для использования в
		веб-приложениях. Они предназначены для использования с JavaScript и для
		широкого использования в веб-приложении и содержат элементы, обычно
		требуемые в современных веб-приложениях. Некоторые из наиболее
		распространенных фреймворков CSS:
			

			- Bootstrap
				
				https://www.w3schools.com/bootstrap4/

			- SASS
				
				https://sass-lang.com/

			- Foundation
				
				https://en.wikipedia.org/wiki/Foundation_(framework)

			- Bulma
				
				https://bulma.io/

			- Pure
